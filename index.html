Come run with us in Louisville's beautiful Iroquois Park

Each year, we host several fun running events including the Wally Bright Memorial Bake-off Run in May, the Toys for Tots Poker Run in August, and the Thanksgiving Day Run in November. Check our events page for a full listing.

In partnership with Swag's Sport Shoes, we also offer summer track sessions at Iroquois High School. Sessions are Tuesday nights during May through August. For more information, visit Swag's website or Facebook page.
